# REACT DESIGN PATTERNS CHALLENGE
This repository contains all my solutions to the react design patterns challenge, displaying my understanding of what I learned from tapascript on YouTub

To clone this repository:
```
git clone <repo link>
```

Navigate to any of the tasks
```
cd day-01-product-list
```

***Make sure you have node js and pnpm installed on your system.***

Install the dependencies with pnom
```
pnpm  install
```

Run the dev server
```
pnpm dev
```

React 19, TypeScript, TailwindCSS, daisyUI and vite were used across the board.
